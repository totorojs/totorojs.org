The totoro family

  • totoro

    Client. As a user, this is the only thing you need to know.

  • totoro-server

    Server of totoro. Any browser that is connected to this server can be used to run test.

  • browsers

    A simple browser manager. It can detect available browsers on the device automatically, start them and visit specified URL, and restart them when neccessary.

  • totoro-log-analyse

    Totoro server log analyzer. It uses mongodb to store data. It supplies some useful info about how totoro be used, such as the overview or recent usage.

For now the default test server is for Alibaba Group only. Whether or not to provide a public server is still under consideration. If the demmand is strong, please let us know.
